About

takushi.us is a hobby project of mine for the purposes of:
  • Hosting a personal blog
  • Learning fullstack development skills
  • Demonstrating skills relevant to my career
Source code can be found at https://github.com/DJTakushi/takushi.

Project Details

Param Value Comments
Backend Django 4.0
  • Python 3.8
  • Django 4.1 was originally used, but had some problems with whitenoise
  • Whitenoise for hosting a few small static files
Hosting AWS
  • Beanstalk
  • Amazon Linux 2 running Python 3.8
  • Database: Amazon RDS running Postgres on a db.t4g.micro
  • Storage: S3 (but my blog has traditionally linked to imgur/flikr uploads)
  • AWS Certificate Manager
I was using Heroku for free to get started, but once I realized that https isn't supported on the free tier, I decided it was high time to learn some AWS. I have mixed feelings about relying on Amazon and designing around their proprietary environment, but it's a good skill to learn and I think I can figure out how to use a VPS later if I'm not satisfied wtih AWS.
Blog Jekyll Also hosted on github pages at https://djtakushi.github.io/dblog/, but I pulled this into takushi.us to increase my site's content. I have to build the static pages on my machine every time I update it, but that's tolerable for now.
Front End Many
  • Bootstrap 4.0
  • JQuery
  • particle.js by marcbruederlin
  • Ajax - required for particle

Further Improvements

  • Update blog for friends & family
  • Client single-page application that accesses djangoTask through API
  • Client applet in home page that make some novel API calls (weather data in various locations, Chicago city data, images from public camers, stock portfolio, etc.)
  • Rental property calculator that can save data to server
  • Donation feature so I can better understand adding a payments feature
  • djangoTask - flesh this out more to be something like JIRA or Evernote
